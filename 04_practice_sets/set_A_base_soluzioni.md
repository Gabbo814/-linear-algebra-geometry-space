# Soluzioni – Practice Set A (outline)

1–2) Riduzione di [A|b]; confronta rank(A) e rank([A|b]); concludi compatibilità e tipo di soluzione (unica/infinita).
3) N(A): risolvi Ax=0 → x1=x2=x3; base {(1,1,1)}; verifica rank(A)+nullità(A)=n.
4) La matrice è una permutazione ciclica: ker(T)={0}, Im(T)=R^3 ⇒ isomorfismo.
5) q1=u1/||u1||; u2' = u2 − ⟨u2,q1⟩ q1; q2 = u2'/||u2'||; P_U=Q Q^T; calcola P_U v.
6) det=1·3−(−2)·1=5 ⇒ invertibile; A^{-1} via Gauss–Jordan.
7) Caratteristico λ^2−3λ+2=0 ⇒ λ=1,2; autovettori come da (A−λI)x=0; diagonalizzabile (basi distinte).
8) Usa vettore direzione di r e normale di π; verifica condizioni (⟨d,n⟩=0 ⇒ r⊥π; d∈π ⇒ r⊂π; altrimenti).
9) dist=|2·1−1·2+2·3−5|/√(2^2+(-1)^2+2^2)=|2−2+6−5|/√9=|1|/3=1/3.
10) Matrice Q=[[3,2],[2,1]]; autovalori λ=2±√5 (segni opposti) ⇒ iperbole; assi tramite autovettori.
