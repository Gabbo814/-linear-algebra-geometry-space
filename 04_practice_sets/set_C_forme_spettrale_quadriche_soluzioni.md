# Soluzioni – Practice Set C (outline)

1) A=[[4,3],[3,5]]; autovalori dalla caratteristica; Q da autovettori ortonormali; segni ⇒ ellisse; rotazione dagli autovettori.

2) Segnatura (2 positivi, 1 negativo, 0 nulli): (2,1,0); Sylvester: esistenza di forma diagonale congruente con stessa segnatura.

3) Congruenza: esiste P invertibile con P^T A P = D (diagonale con ±1, 0) secondo Sylvester.

4) A simmetrica ⇒ ortogonale diagonalizzazione; Q da autovettori ortonormali; Λ diag degli autovalori; ellisse allineata con assi di Q.

5) SVD: M=UΣV^T con Σ=diag(3,1); condizionamento κ=σ_max/σ_min=3; discusso effetto su stabilità di Ax=b.

6) Ellissoide: x^2/1 + y^2/1/2 + z^2/1/3 = 1 (riporta correttamente i semiassi: a=1, b=1/√2, c=1/√3); piani di simmetria coordinati.

7) Paraboloide: sezioni z=c → cerchi x^2+y^2=c (se c>0); linee di livello circolari; asse lungo z.

8) Cono doppio: sezioni z=c danno cerchi (c≠0) o punto (c=0); simmetrie assiali.

9) Centro: c=−(1/2)A^{-1} l; sostituzione y=x−c; forma canonica dalla parte quadratica diagonalizzata.

10) Angolo tra piani: cosθ = |n1·n2|/(||n1||·||n2||), n_i=(a_i,b_i,c_i).
